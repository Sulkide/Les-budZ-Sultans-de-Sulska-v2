README Les budZ (Sultans de Sulska)


Introduction :

Dans le pays du Sulska, le vaillant Sulkide a trouvé le système D, lui permettant d'écraser le monde en 2 dimensions.
Cette découverte lui ouvrit les portes à une aventure dénuée de lois physiques cohérentes.


Contrôles :

Q - gauche
D - droite
Z - avant (uniquement mode 3d)
S - arrière (uniquement mode 3d)

Entrée - Changer de dimension
Espace - Sauter
Shift - Dash
Flèches directionelles - Viser
E - Tirer une flèche


Mouvement :

Sulkide peut dash dans toutes les directions lorsqu'il est en 2D, mais uniquement dans la direction dans laquelle il se déplace en 3D.
Le dash est une aide, aucun niveau ne nécessite de dash pour en venir à bout. Pour une expérience plus exigeante, vous pouvez vous en passer.

Sauter contre un mur en maintenant la direction permet de walljump.


Ennemis :

Sulkide peut tirer des flèches à tout moment pour anéantir les ennemis sur son chemin. Il peut également leur sauter dessus.
Il faut obligatoirement viser (flèches directionelles) avant de tirer.
